📘 SafeNest UI — Product Documentation
🧠 Overview

SafeNest UI, Godot Engine 4.x için geliştirilmiş, mobil cihazlarda UI yerleşimini otomatik olarak güvenli alanlara (safe area) adapte eden hafif ve üretim odaklı bir editor eklentisidir.

Amaç:
Mobil UI problemlerini (notch, farklı ekran oranları, taşan HUD) tek tıkla çözmek.

🎯 Problem Definition

Mobil platformlarda UI geliştirme sırasında karşılaşılan temel problemler:

Notch / punch-hole alanları
Farklı aspect ratio’lar (16:9, 20:9 vs.)
UI elementlerinin ekran dışına taşması
Her cihaz için manuel düzenleme ihtiyacı

Bu problemler özellikle:

Solo developer’lar
Mobile-first projeler
Hızlı prototipleme süreçleri

için ciddi zaman kaybına yol açar.

💡 Solution

SafeNest UI, bu problemi aşağıdaki şekilde çözer:

Safe area verisini runtime’da okur
Seçilen UI node’una otomatik layout uygular
Tek tıkla mobil uyumlu hale getirir
Debug overlay ile sonucu görselleştirir
🚀 Core Features (V1)
1. Safe Area Service
Cihazın güvenli alan bilgilerini alır
Runtime ortamında çalışır
2. Auto UI Layout Fix (Core Feature)
Seçilen Control node’una uygulanır
Anchor + margin otomatik ayarlanır

Primary Action:

“Apply Mobile Safe Layout”

3. Debug Overlay
Safe area sınırlarını görselleştirir
Açılıp kapatılabilir
4. Editor Tool Panel
Editor içinde dock panel
Node seçimi + tek tık uygulama

🎮 Usage
1. Addon’u aktif et
Project Settings → Plugins → Enable
2. UI node seç
Control tabanlı bir node seç
3. Tool panelden uygula
“Apply Mobile Safe Layout” butonuna bas
4. Debug overlay aç (opsiyonel)
Safe area’yı görselleştir
